  • Image Deletion: Remove sent images.
  • Emoji Reactions: React to messages with emojis.
  • Private Chat Images: Send images in private chat.
  • Groups: Create groups and chat together.
  • More Bug Fixes & Stability Improvements.
  • World Chat: Chat with everyone in real-time.
  • Private Chat: Find and chat with specific users.
