How to Use (v1.0.51)

  • Option feature added: Easier app navigation.
  • Image react added: React to images with emojis.
  • Reply function added: Easily reply to specific messages.
  • Users Last Seen: See when users were last online.
  • Message seen bug fixed: Improved message read status.
  • New themes added: Customize your chat experience.
  • World Chat: Chat with everyone in real-time.
  • Private Chat: Find and chat with specific users.
  • If a username is red, it means they sent you an unread message.
  • If a username is black, it means there are no new messages.
  • If a username is green, the user is online.
